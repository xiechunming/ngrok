# ngrok
10分钟搞定ngrok内网映射外网 ngrok国内服务器搭建 微信本地测试环境搭建


  大家都知道，微信后台开发服务器需要在公网能访问，本地开发需要测试的时候如果没有公网实现不了，ngrok帮我们解决了这个问题，他可以把本地的服务器地址映射的公网上面，ngrok自身的服务器在国外，没时候会访问过慢，不过没关系，国内的好多服务器都提供ngrok服务，（http://www.ittun.com/）今天就以它为例，10分钟教你内网映射到外网。
